# Neural-Forge-Ai API

With **Neural-Forge-Ai API**, you can run your own instance of **Neural-Forge-Ai** and leverage it to 
train models on Hugging Face Spaces infrastructure (local training support coming soon). 
This API is designed to work with **Neural-Forge-Ai** compatible models and datasets, providing a 
simple and efficient interface for training models with minimal configuration.

## ðŸš€ Getting Started

To get started with the **Neural-Forge-Ai API**, first install `neural-forge-ai` 
as discussed in the **Running Locally** section, and then start the application using:

```bash
$ neural-forge app --port 8000 --host 127.0.0.1
```

You can then access the API reference at `http://127.0.0.1:8000/docs`.

## Example Usage

```bash
curl -X POST "http://127.0.0.1:8000/api/create_project" \
     -H "Content-Type: application/json" \
     -H "Authorization: Bearer hf_XXXXX" \
     -d '{
           "username": "punith",
           "project_name": "my-neural-forge-ai-model",
           "task": "llm:orpo",
           "base_model": "meta-llama/Meta-Llama-3-8B-Instruct",
           "hub_dataset": "argilla/distilabel-capybara-dpo-7k-binarized",
           "train_split": "train",
           "hardware": "spaces-a10g-large",
           "column_mapping": {
               "text_column": "chosen",
               "rejected_text_column": "rejected",
               "prompt_text_column": "prompt"
           },
           "params": {
               "block_size": 1024,
               "model_max_length": 4096,
               "max_prompt_length": 512,
               "epochs": 1,
               "batch_size": 2,
               "lr": 0.00003,
               "peft": true,
               "quantization": "int4",
               "target_modules": "all-linear",
               "padding": "right",
               "optimizer": "adamw_torch",
               "scheduler": "linear",
               "gradient_accumulation": 4,
               "mixed_precision": "fp16",
               "chat_template": "chatml"
           }
         }'
```
